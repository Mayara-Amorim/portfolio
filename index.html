Tecnologias

Um pouco sobre as principais stacks tecnológicas que eu utilizo e tenho contato

Lógica de Programação


Encadeamento de pensamentos para atingir determinado objetivo. Descrição de como uma seqüência de instruções, devem ser seguidas para se cumprir uma determinada tarefa, ou seja: seqüência Lógica ou Algoritmos. Eu poderia usar um papel, mas prefiro uma IDE :)

Python


Foi minha primeira linguagem de programação. Com Python exercito minha lógica. Por ser uma linguagem de alto nível, faz com que eu me concentre mais nos algoritmos do que na sintaxe em si.
Está no repertório para meu TCC, com certeza!

HTML5, CSS3 e JS


Eu utilizei essas essas tecnologias para a criação dessa e outras páginas. Com certeza a nata do Desenvolvimento Front-end!

Banco de Dados Relacional


Já ultilizei SQL (Structured Query Language) em diversos SGBDs: desde Oracle até o H2 da IBM.
Tenho conhecimentos desde a criação do modelo lógico/ relacional até a realização de normalizações. Estou estudando a linguagem estruturada da Oracle: PL/ SQL

JAVA


A flataforma Java é a minha preferida para o backend: ela é madura e rebusta, com uma excelente documentação. Estou longe de saber tudo.
É minha dose diária de desafio! Com Java construo desde projetos totalmente Orientados á Objetos até APIs RESTfull.

Spring


O framework mais amado do mundo Java. Sim, o casamento perfeito! É uma sensação maravilhosa fazer um CRUD com Spring, muita produtivadade e simplicidade. No Java com Spring, nunca me esqueço do JPA/Hibernate e Maven. É essencial, meus caros(as)!!!
